1. the top node is called root,and its below nodes are children
2. leaf node - the node which does not have children is called leaf node
3. subtree - the node and beneath nodes are called sub trees
4. ancestor - all the parent and above nodes

types of binary tree
1. full binary tree - every node has 2 or 0 children
2. complete binary tree
    1.all levels are completely filled expect the last level
    2. the last level has all nodes has left as possible
3. perfect binary tree - all leaf nodes are at same level
4. balanced binary tree - height should be maximum of log n
5. degenerate tree - its a skew tree looks like a straight line
